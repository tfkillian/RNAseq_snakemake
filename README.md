# RNAseq_snakemake

This Snakemake pipeline automates a RNAseq workflow utilizing bash command line tools, fastqc trimmomatic, hisat2 and htseq-count. 

## Dependencies

    - conda
    - snakemake
    - fastqc
    - trimmomatic-PE
    - hisat2
    - htseq-count
